# mdBook Summarizer

A better [mdbook-auto-gen-summary](https://github.com/cococolanosugar/mdbook-auto-gen-summary) for me.

## Goal

Extract more information from content to title.

## Accomplished Features

- Generate `SUMMARY.md` in format.
- Ignore non-chapter files.
- Sort chapters by filenames.

## Unimplemented Features

- Become a [mdBook preprocessors](https://rust-lang.github.io/mdBook/for_developers/preprocessors.html).
   Need to run everytime you need it.
- LLM powered chapter content digest to title.
